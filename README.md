# CarbonFlow

안녕하세요. (주)하나루프 프론트엔드엔지니어 지원자 이예빈입니다.
탄소 배출량 관리 SaaS 플랫폼 개발자 채용 과제입니다.

## 1. 프로젝트 소개

과제용 원본 활동 데이터(전기, 원소재, 운송)를 기반으로 배출계수를 매칭해 PCF(Product Carbon Footprint)를 계산하고, Scope 2 / Scope 3, 활동 유형별 배출량, 월별 추이를 시각화하는 대시보드입니다.
실행 시 회사에서 제공한 원본 데이터로 표시된 후 add data를 통해서 원하는 데이터로 확인할 수 있습니다.

## 2. 실행 방법

```bash
git clone 저장소주소
cd carbonflow
npm install
npm run build
npm run start
```
